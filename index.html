About Me

Hi, I'm Chris Stirneman, an aspiring web developer with a passion for creating clean, functional, and visually appealing websites. I'm 24 years old and I live in Louisville, KY with my girlfriend of 3 years and our dog. I graduated in 2018 from Bellarmine University with a Bachelor’s degree in Exercise Science and initially pursued a Doctorate in Physical Therapy. However, I realized this wasn’t my calling and decided to pivot toward software engineering. In 2024, I earned a Bachelor’s degree in Software and Data Engineering, also from Bellarmine University. Since then, I’ve dedicated myself to honing my skills in front-end development, focusing on HTML, CSS, and JavaScript, and have recently started learning React. I enjoy bringing design concepts to life and crafting responsive, user-friendly experiences. I’m constantly exploring new technologies and refining my coding practices to deliver the best possible results.

In addition to my technical skills, I have experience working in customer relations, which has strengthened my communication skills and taught me the importance of understanding user needs. I’m excited to transition into a full-time role where I can contribute to meaningful projects, collaborate with talented teams, and grow my expertise in web development. Let's connect and see how I can help bring your ideas to life!

Projects

Customer Service App

An app I built for the customer service representatives at NetLaw (the place I am currently employed). This app aims to bring more efficiency to our service department by allowing representatives to quickly copy and paste responses to FAQ's, find and download commonly used resources, and access a repository of information that is vital to performing their day-to-day tasks.

Technologies used: HTML, CSS, JavaScript

Quiz App

This quiz app is designed for users to create, review, edit, and save custom quizzes. Featuring a user authentication system, it securely stores user data and quiz information using MySQL. The backend is powered by Node.js with Express for efficient server management and bcrypt for password encryption. This app provides a streamlined experience for quiz creation and review, supporting personalized learning and content organization.

Technologies used: HTML, CSS, JavaScript, Node.js, Express, MySQL

Simon Game

This project features a simon memory gamewhere the player must repeat an increasingly complex sequence of colors and sounds. The game displays a pattern on a set of colored buttons, and the player must replicate that pattern in the correct order. With each round, the sequence becomes longer, challenging the player's memory and attention.

Technologies used: HTML, CSS, JavaScript

To-Do List

I created a to-do list application to help users organize their tasks efficiently and stay on top of their daily goals. The app features sections for 'To-Do' and 'Completed' tasks, allowing users to easily add, edit, and check off items as they complete them. It also includes options to set due dates, prioritize tasks by urgency, and keep track of progress.

Technologies used: HTML, CSS, JavaScript

Memory Game

I developed a memory matching game as a fun and interactive way to challenge users’ concentration and memory skills. The game presents a grid of cards, each hiding an image. Players flip two cards at a time, trying to find matching pairs. If the cards match, they stay revealed; if not, they flip back over, encouraging the player to remember their locations. The game ends when all pairs are matched.

Technologies used: HTML, CSS, JavaScript

Weather App

A weather app that uses the OpenWeather API to provide real-time weather information for any city worldwide. Users can search for a location and instantly view the current temperature, weather conditions, humidity, and wind speed.

Technologies used: HTML, CSS, JavaScript
